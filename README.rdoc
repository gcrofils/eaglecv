= EagleCV

EagleCV is a collaborative tool to manage cv.

== Install and use

=== Install EagleCV and setup your application

  git clone git://github.com/gcrofils/eaglecv.git
  cd eaglecv
  ...
  
  cd config
  cp application.yml.sample application.yml
  cp database.yml.sample database.yml
  cp development.rb.sample development.rb
  
  Setup the config files
